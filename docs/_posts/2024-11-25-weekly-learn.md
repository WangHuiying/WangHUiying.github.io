1. 设置 Prettier 为默认格式化工具
	1.	打开 VS Code 设置：
  	•	点击右下角齿轮图标 → 选择 Settings。
  	•	或按快捷键 Cmd+。
	2.	搜索 default formatter，然后设置 Prettier 为默认格式化工具：
	  •	找到 Editor: Default Formatter，选择 esbenp.prettier-vscode（Prettier 插件）。
	3.	启用自动格式化：
	  •	搜索 format on save，勾选 Editor: Format On Save。
